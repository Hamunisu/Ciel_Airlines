# Ciel Japan Airlines

## 概要
***Ciel Japan Airlines***（バーチャルエアライン）のパイロット向け便名検索サイト

- **サイト構造**:
WebAssembly化したRustのコアロジックをJavaScript（フレームワーク非依存）から呼び出しブラウザ内で便名の検索処理を完結


## 公開サイト
[Ciel Japan Airlines 公式サイト](https://hamunisu.github.io/Ciel_Airlines/)


## ⚒️技術スタック
- **フロントエンド**: HTML / CSS / JavaScript(Vanilla JS -- フレームワーク非依存)
- **コアエンジン**: Rust / WebAssembly(Wasm)

## 🧩コアロジックについて
検索ロジックおよびデータ構造についての詳細なソースコードは、透明性確保のため以下の別リポジトリにて一般公開されています。

[Ciel_Core](https://github.com/Hamunisu/Ciel_Core)
